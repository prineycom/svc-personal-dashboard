# Firefly III для финансов без банк-синхронизации

Выбран Firefly III как self-hosted финансовый трекер. Zen Money отклонён: API ограничен (только `/v8/diff/` и `/v8/suggest/`), нет MCP, SaaS (данные не у нас).

## Why Firefly III

- Полный REST API
- 3 MCP-сервера (рекомендую `fabianonetto/mcp-server-firefly-iii` с 66 инструментами)
- Docker, активно развивается
- Банк-синхронизация не нужна — ручной ввод + CSV/Data Importer

## Consequences

- Финансовые данные — самые чувствительные, self-host приоритетнее чем принцип «не дублировать хорошие внешние сервисы».
- Ручной ввод — компромисс, принят сознательно.
