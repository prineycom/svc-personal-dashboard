# Гибридная архитектура: внешние сервисы где лучше, self-host для остального

Personal Dashboard использует гибридный подход: внешние SaaS-сервисы (Google Calendar) для направлений, где нет качественного self-host аналога, и self-hosted решения для всего остального. Hermes Agent — единая точка входа через MCP. Ежедневник — не отдельный сервис, а оркестрационный сценарий (утренний/вечерний cron).

## Considered Options

- **A)** Полностью self-hosted — все сервисы на Pi, даже если нет подходящего решения
- **B)** Полностью SaaS — TickTick, Google, Todoist и тд
- **C)** Гибрид — self-hosted приоритет, SaaS только как исключение ✅

## Consequences

- Каждый новый direction сначала оценивается на self-host альтернативы. SaaS только если self-host варианта нет или она некачественная.
- Google Calendar — единственное SaaS-исключение на данный момент.
